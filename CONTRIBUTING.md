# Contributing

This is a lightweight single-page map game. The safest way to make changes is to keep a versioned backup before editing `index.html`.

## Suggested workflow

1. Copy the current `index.html` to a versioned file, for example `themapgameofficial_v77.html`.
2. Make the change in `index.html`.
3. Test the game in a browser.
4. Check these areas before committing:
   - map modes still switch correctly;
   - quiz starts and advances;
   - popups and hover labels still work;
   - capital markers still toggle;
   - the dispute/identity panel still opens and closes;
   - mobile layout is still usable.
5. Record the change in `CHANGELOG.md`.

## Style guidance

- Keep dispute and identity wording neutral.
- Avoid implying endorsement of any political claim.
- Prefer short educational notes over long explanations inside the game UI.
- Keep labels concise so they do not clutter the map.
