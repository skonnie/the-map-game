# Root Folder Guide

When you open the GitHub repository, the root folder is the first file list shown under the branch selector.

Correct root-level URL:

```text
github.com/YOUR-USERNAME/YOUR-REPO/blob/main/index.html
```

Wrong nested URL:

```text
github.com/YOUR-USERNAME/YOUR-REPO/blob/main/some-folder/index.html
```

For this project, `index.html` and `un-wpp-population-latest.json` must be at root level.
