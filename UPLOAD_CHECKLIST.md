# Upload Checklist

Before committing to GitHub, confirm the root of the repo contains:

```text
index.html
un-wpp-population-latest.json
.nojekyll
README.md
```

The game will still run with only:

```text
index.html
un-wpp-population-latest.json
```

After upload:

1. Open `index.html` on GitHub.
2. Confirm it starts with `<!doctype html>`.
3. Go to Settings → Pages.
4. Choose Deploy from a branch.
5. Select `main` and `/ root`.
6. Visit the GitHub Pages URL after deployment finishes.
